HTML Elements

keyword : html elemen/tag

  • tag html adalah nama elemen yang dikelilingi oleh angle brackets ( seperti <h5> )

  • kalo 1 elemen html adalah semuanya mulai dari start tag sampai ke end tag.

    • Start Tag Content End Tag
    <h1> hello </h1>
    <br> tidak ada tidak ada

  • ada bbrp elemen yg tidak punya content ( seperti <br> )

  • elemen html bisa bersarang, jadi didalam elemen bisa terdapat elemen2 lainnya.

  • elemen html tidak case sensitive, tapi direkomendasikan ditulis dalam lowercase

    • Contoh :
    <body> adalah elemen yang mendefinisikan document's body
    doi punya start tag <body> dan end tag </body>
    di dalam doi ada lagi 2 elemen, seperti <h1> dan <p>
